A retro analog photo booth web application that captures four sequential photos and displays them as a downloadable strip with vintage film effects.


##  Features

-  **Real-time camera preview** with professional viewfinder overlay
-  **High-contrast B&W filter** for authentic photo booth aesthetic ‚ö´‚ö™
-  **4-photo sequence capture** with countdown timer
-  **Flash effect** on each capture
-  **Vintage photo strip** with film grain, dust, and scratches
-  **Download as JPEG** - High-quality strip export
-  **GSAP animations** - Smooth, elastic bounce effects
-  **Responsive design** - Works on desktop and mobile
-  **Retro UI** - Film grain, scanlines, and analog overlays

##  Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** GSAP
- **Camera:** Browser MediaDevices API
- **Canvas:** HTML5 Canvas for image processing


##  Usage

1. **Landing Page** - Click "BEGIN SESSION" to start
2. **Camera Preview** - Allow camera permissions when prompted
3. **Capture** - Click "CAPTURE" to start the 4-photo sequence
4. **Countdown** - Pose for each photo (3-second countdown)
5. **Photo Strip** - View your retro strip with vintage effects
6. **Download** - Save your strip or retake photos


##  Design Features

### Camera Interface
- Professional viewfinder with corner brackets
- Center crosshair for framing
- Rule of thirds grid overlay
- Real-time system info (REC indicator, timestamp, ISO)
- High-contrast B&W filter with adjustable settings

### Photo Strip
- Classic 4-frame vertical layout
- Vintage paper texture
- Film grain overlay
- Dust and scratch effects
- Date stamp footer
- GSAP elastic bounce animation
